Passo a passo:

 - Entrar no repositório;
 - Usar o git add . (Para selecionar todos os arquivos da pasta);
 - Verificar o status para ver se tudo está indo bem (git status);
 - Hora de commitar! git commit -m "Aqui vai o comentário do commit";
 - Agora é preciso dar um push: git push origin main
 - E enviamos o git local para o remoto!
